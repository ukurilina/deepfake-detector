# Frontend MVP (Vanilla JS)

В этой папке находится статический веб-фронтенд для FastAPI backend из `app/main.py`.

## Что уже реализовано

- Главная страница с 2 действиями:
  - `Проверить файл на дипфейк` (активно)
  - `Защитить мой файл от создания дипфейка` (заглушка)
- Страница проверки:
  - загрузка файла с компьютера для `Фото` и `Видео`
  - проверка по URL с обязательным выбором типа контента
  - выбор модели (через `/models?content_type=...`) для обоих режимов
  - модели фильтруются по выбранному типу контента
  - `Аудио` отображается как `Скоро`
  - задание порога
  - запросы к backend:
        - `POST /detect` для файла (`multipart/form-data`, поле `file`, query `content_type`)
        - `POST /predict/url` для URL (`application/json`, поле `content_type`)
  - отображение результата (процент, вердикт, метка, уверенность, модель, порог)
  - состояния загрузки и ошибок
- Заглушка на странице защиты:
  - загрузка/преобразование файла пока недоступны

## Локальный запуск (Windows)

Из корня проекта (`deepfake_detector`) запустите backend:

```bash
python -m uvicorn app.main:app --reload
```

В другом терминале запустите статический сервер фронтенда:

```bash
cd frontend
python -m http.server 3000
```

Откройте в браузере:

- `http://127.0.0.1:3000`

## Настройка URL backend

Отредактируйте `frontend/assets/js/config.js`:

- `API_BASE_URL`: базовый URL backend (по умолчанию `http://127.0.0.1:8000`)
- `MAX_FILE_SIZE_MB`: лимит размера файла на клиенте
- `SUPPORTED_MEDIA`: допустимые расширения и MIME по типам (`photo`, `video`, `audio`)
